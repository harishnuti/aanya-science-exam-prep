🧩 QUESTION DESIGN

What Makes These Brain Drainers:

✅ SIMILAR DISTRACTORS
   All 4 options are plausible at first glance
   Must read carefully to distinguish correct answer
   Tests understanding, not guessing

✅ MULTI-CONCEPT
   Combine 2-3 concepts together
   Not just simple recall
   Requires deeper understanding

✅ TRICKY WORDING
   Questions worded to catch misconceptions
   All wrong answers appeal to natural (but incorrect) thinking
   Teaches WHAT NOT to think

✅ SCENARIO-BASED
   Real-world applications
   Tests ability to apply knowledge
   More challenging than definitions

✅ PSLE-AUTHENTIC
   Based on real PSLE question patterns
   Matches actual exam difficulty
   Prepares thinking style for real exam

───────────────────────────────────────────────────────────────────────

🎯 EXPECTED DIFFICULTY

Scoring Guide:
  🟢 80%+ = "You have a sharp mind!" (Excellent)
  🟡 70-79% = "Great job! These were PSLE-level!" (Good)
  🟠 50-69% = "Brain drainers are meant to be hard!" (OK)
  🔴 <50% = "Keep practicing!" (Try again)

NOTE: These are INTENTIONALLY HARD!
  - Expected score: 60-80% (NOT 100%)
  - Getting 80%+ shows true understanding
  - Scoring less than 100% is EXPECTED and GOOD!
  - Brain drainers teach, not test perfection

───────────────────────────────────────────────────────────────────────

✅ SAFETY FEATURES

✓ Next Button Disabled Until Answered
  - Prevents accidentally skipping questions
  - Ensures user reads and thinks about each question

✓ Double-Count Guard
  - If user goes back and re-answers, score doesn't increment twice
  - Keeps scoring fair and accurate

✓ Session State Tracking
  - All answers stored with confidence levels
  - Full answer history available in results
  - Detailed explanations for each question

✓ Confidence Rating
  - Tracks whether user was guessing (1) or sure (5)
  - Helps identify areas of uncertainty
  - Shown in results summary
