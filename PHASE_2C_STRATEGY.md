| **Non-flowering** | Spores, dispersed by water/wind | Pages 18-19 |
| **Seed Germination** | Needs WOW: Water, Oxygen, Warmth | Page 17 |

### **Chapter 2: Water Cycle**
| State | Properties | Textbook Ref |
|-------|-----------|---------|
| **Solid (Ice)** | Melts at 0°C | Pages 27, 31 |
| **Liquid (Water)** | Normal form, boils at 100°C | Page 28 |
| **Gas (Vapour/Steam)** | Invisible in air; visible when boiling | Pages 28-29 |
| **Melting** | Solid → Liquid | Pages 33-34 |
| **Boiling** | Liquid → Gas | Page 35 |
| **Freezing** | Liquid → Solid | Page 35 |
| **Condensation** | Gas → Liquid | Page 35 |
| **Evaporation** | Liquid → Gas (without boiling) | Page 37 |
| **Water Cycle** | Evaporation → Condensation → Precipitation → Collection | Pages 43-44 |

### **Chapter 3: Plant Transport**
| System | Function | Details | Textbook Ref |
|--------|----------|---------|---------|
| **Xylem** | Water/Minerals | From roots to leaves, appears as wood | Pages 55-57 |
| **Phloem** | Food (glucose) | From leaves to all parts, living cells | Pages 57-58 |
| **Root Hairs** | Absorption | Increase surface area for water uptake | Page 54 |
| **Transpiration** | Water loss | Water vapor escapes from leaves | Page 59 |
| **Removal Effects** | No xylem = wilts, No phloem = starves | Pages 60-61 |

### **Chapter 4: Respiratory & Circulatory**
| System | Component | Function | Textbook Ref |
|--------|-----------|----------|---------|
| **Respiratory** | Nasal cavity | Warms, moistens air | Page 69 |
| | Trachea | Carries air to lungs | Page 69 |
| | Lungs | Gaseous exchange | Pages 69-71 |
| | Diaphragm | Aids breathing (contracts/relaxes) | Page 70 |
| **Circulatory** | Heart | Pumps blood | Pages 75-76 |
| | Arteries | Carry oxygenated blood away | Page 76 |
| | Veins | Carry deoxygenated blood back | Page 76 |
| **Pulse** | Heartbeat | Felt at wrist/neck | Page 76 |
| **Activity Effect** | Exercise | Increases breathing & heart rate | Page 80 |

### **Chapter 5: Electrical Systems**
| Concept | Details | Textbook Ref |
|---------|---------|---------|
| **Circuit Components** | Battery, bulb, wires, switch | Pages 93-95 |
| **Closed vs. Open** | Complete path = light; broken = dark | Pages 94-96 |
| **Current Flow** | Electrons flow; measured in Amperes | Page 97 |
| **Conductors** | Metals (copper, aluminum, iron, silver) | Page 98 |
| **Insulators** | Rubber, plastic, wood, glass | Page 98 |
| **Circuit Symbols** | Standard diagrams (battery, bulb, switch, ammeter) | Pages 102-103 |
| **Safety** | Never touch wet appliances; dry hands only | Pages 101-102 |

### **Chapter 6: Series Circuits**
| Parameter | Effect | Formula | Textbook Ref |
|-----------|--------|---------|---------|
| **More Batteries** | More voltage → More current → Brighter bulbs | V = I × R | Pages 108-110 |
| **More Bulbs (series)** | More resistance → Less current → Dimmer bulbs | R_total = R₁ + R₂ | Pages 110-113 |
| **Current in Series** | Same throughout the circuit | Fixed path | Page 109 |
| **Voltage in Series** | Divides among components | Less per bulb | Page 110 |
| **Circuit Breakers** | Prevent overheating/fire | Trip when current too high | Page 113 |
| **Safety Practices** | Avoid overload, use certified devices | Prevent accidents | Pages 113-115 |
| **Energy Conservation** | Turn off devices, use public transport | Reduces fossil fuels | Pages 114-115 |
